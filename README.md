# AWS Serverless Data Analytics Pipeline – E-Commerce Sales Analysis

## Overview
This project implements a fully serverless data analytics pipeline on AWS to ingest, transform, analyze, and visualize e-commerce sales data from both API and batch sources.

## Architecture
API Gateway → Lambda → S3 → AWS Glue → Athena → Power BI → CloudWatch

## AWS Services Used
- Amazon API Gateway
- AWS Lambda
- Amazon S3
- AWS Glue
- Amazon Athena
- Amazon CloudWatch
- Power BI

## Features
- Serverless data ingestion from REST API
- ETL processing using AWS Glue
- Optimized Parquet storage
- SQL-based analytics using Athena
- Interactive Power BI dashboards
- Monitoring and logging using CloudWatch

## Dashboards
- Total Revenue
- Units Sold
- Average Review Score
- Revenue by Category
- Monthly Sales Trends
- Top Products Analysis

- Serverless data engineering
- Data lake architecture
- ETL pipelines
